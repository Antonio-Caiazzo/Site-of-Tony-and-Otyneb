Descrizione SQL


        In informatica, SQL (Structured Query Language) è un linguaggio standardizzato per database basati sul modello relazionale (RDBMS), 
        progettato per le seguenti operazioni:

        creare e modificare schemi di database (DDL = Data Definition Language);
        inserire, modificare e gestire dati memorizzati (DML = Data Manipulation Language);
        interrogare i dati memorizzati (DQL = Data Query Language);
        creare e gestire strumenti di controllo e accesso ai dati (DCL = Data Control Language).
        A dispetto del nome, non si tratta perciò di un semplice linguaggio di interrogazione: alcuni suoi sottoinsiemi, infatti,
        permettono di creare, gestire e amministrare database.
